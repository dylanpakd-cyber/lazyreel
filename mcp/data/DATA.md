# Data sources

This folder holds the redistributable data the skills draw on. Everything here is either original to this project or permissively licensed.

## analyzed-videos.json

Original work. These are representative teardowns of short form video patterns that perform, authored in LazyReel's own framework language. They are not scraped TikTok content and contain no real user data. Safe to redistribute. Grow this file freely.

## trending-hashtags.csv

Real TikTok trending hashtag rankings for 2022 through 2025 (1,830 rows: tag, year, rank, posts). This is aggregate ranking data, not individual user content.

Source: ronantakizawa/tiktok-trending-hashtags on Hugging Face.
License: MIT.

## insights.json and corpus-stats.json

Generated by the decode pipeline (see ../pipeline/README.md). These are derived statistics from a real scrape: counts and average views by hook pattern and niche, plus the computed corpus counts the MCP and site read. They contain no source video text, so they are safe to redistribute. The numbers are computed from the data, never hand typed.

## What is deliberately not here

Raw scraped content (captions and transcripts) lives in data/raw/ and data/decoded/, which are gitignored. That content is TikTok sourced and should not be redistributed. Anyone can regenerate it with their own Apify key by running the pipeline. We ship our analysis, not their content.
